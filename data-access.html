Data & Contact

Access the data, cite the work, get in touch

Open data

AlignED benchmark data is openly available. The data files used by this site are published in the GitHub repository. The full dataset, including raw model outputs, is being prepared for release on the Open Science Framework (OSF).

The dataset includes:

  • Model scores for all six benchmarks (up to 31 models per benchmark)
  • ACARA accuracy and reliability scores for 12 models
  • Item-level responses for all benchmarks
  • Raw model outputs and reasoning
  • Judge scoring with explanations (for diagnostic reasoning scenarios)
  • Temperature robustness data (T=0, 0.5, 1.0)
  • Prompt sensitivity results (4 framings)
  • Confidence probe responses
  • Token usage statistics

The dataset grows as new evaluations are completed. This is an active research project.

Citation

If you use AlignED data in your work, please cite:

AlignED Benchmark (2026). Benchmarking AI performance on professional teaching tasks. https://trgallagher-research.github.io/AlignED-site/

If you cite AlignED in academic publications, please let us know at the address below so we can provide an up-to-date citation and track usage of the benchmark.

Terms of use

AlignED data is provided for research, policy analysis, and educational purposes. When using the data:

  • Cite AlignED appropriately
  • Do not misrepresent scores as "certification" or "approval"
  • Acknowledge limitations noted in About
  • Contact us if you plan to redistribute modified versions

Contact
